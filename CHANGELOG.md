# Changelog

All notable changes to this project will be documented in this file.
This project follows Keep a Changelog and Semantic Versioning.

## [0.1.0] - Unreleased
### Added
- Core data contracts (`IVData`, `IVDesign`) and linear algebra utilities.
- Robust moment covariance estimators (HC0/HC1, cluster, homoskedastic option).
- TSLS estimator and LIML point estimates for workflow support.
- Anderson-Rubin tests and set-valued confidence sets with diagnostics.
- Weak-IV benchmark generators and smoke benchmark runner.
- Expanded docs, tutorials, and examples.
- Unit, integration, and parity tests (ivmodels when available).
