## Reproducibility

All figures in this gallery are generated by the following command:

```bash
python scripts/build_figures.py
```

Each script uses fixed random seeds to ensure exact reproducibility. The plotting style 
is defined in `assets/ivrobust.mplstyle` and applied automatically via `ivr.set_style()`.

!!! tip "Publication-Ready Output"
    Use `ivr.savefig(fig, "path/to/figure", formats=("png", "pdf"))` to export figures 
    in both raster and vector formats suitable for journal submission.
